## Adding / Editing Quotes

Open `src/features/players/header/types.ts` and edit the `QUOTES` array:

```typescript
export const QUOTES: Quote[] = [
  { text: 'The shuttle doesn\'t wait for anyone. Neither does the court.', author: 'Coach Amal' },
  { text: 'Every rally is a new beginning.' },
  // Add more here
]
```

Quote rotates daily (day-of-year mod length). Changes take effect on next deploy.

---

## Adding Announcements

Edit the `ANNOUNCEMENTS` array in `types.ts`:

```typescript
export const ANNOUNCEMENTS: Announcement[] = [
  {
    id: 'ann-001',
    title: 'Court Closed — Long Weekend',
    body: 'The court will be closed on April 22 and 23 for the public holiday.',
    date: '2026-04-18',
  },
]
```

When the array is non-empty, the bell icon in the player header will show a badge.

---

## Adding Sponsors

Edit the `SPONSORS` array in `types.ts`:

```typescript
export const SPONSORS: Sponsor[] = [
  {
    id: 'sponsor-001',
    name: 'Yonex Sri Lanka',
    logoUrl: 'https://example.com/yonex-logo.png',
    websiteUrl: 'https://yonex.com',
  },
  {
    id: 'sponsor-002',
    name: 'Local Sports Store',
    // No logoUrl — name will display as text fallback
  },
]
```

When the array is empty, the sponsors section shows "No current sponsors" instead of blank space.

## Verifying the Changes

### Admin Dashboard (US1)
1. Log in as admin and navigate to `/admin`.
2. Confirm there are no Prev/Next date buttons and no date display text.
3. Confirm the metrics cards load automatically without any user interaction.

### Player Header (US2)
1. Open the root URL `/` (public calendar page).
2. Confirm a quote is visible inside the blue header bar below "THE SHUTTLE".
3. Confirm a bell icon is visible in the header.
4. Click the bell icon — a popover appears (empty state or announcement list).
5. Press Escape — popover closes.
6. Resize to 375 px wide — quote and bell must remain readable, no overlap.

### Sponsors (US3)
1. On the public calendar page, scroll to (or look just below) the blue header.
2. Confirm the sponsors section is visible with an "Our Sponsors" label.
3. With `SPONSORS = []`, confirm a fallback text ("No current sponsors") is shown.
4. With sponsors populated, confirm each card is clickable (if websiteUrl set) and
   shows the logo or name text.
